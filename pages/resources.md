### Useful Sketch Plugins

Craft
: insert images, text and duplicate on the same theme to make mockups faster to make (you should have it from the pre-work already)

Clipboard Fill
: enables you to copy an image, select a shape and paste it there where it will automatically adapt the image to the shape

Icon Font
: (you should have it from the pre-work already)

Magic mirror
: fast mockups, very useful for presentations

Rename it
: enables you to rename many layers / groups with the same name, or in a sequence (name 1, name 2…)

Prism
: automatically creates the color palette from the colors used in your document

Runner
: like a search bar to find a symbol you have creates, a layer, a group…

Measure
: adds red lines on measurements or distances of any element
